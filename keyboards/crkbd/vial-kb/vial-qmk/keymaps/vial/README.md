# My custom config file

## Настройка и установка
1. Используй основной readme для установки ключевых пакетов
2. Компиляция: `kb=crkbd kr=rev4_1/standard km=vial make vial-qmk-compile`
3. Загрузка прошивки - в vial нужно забиндить кнопку BootLoader 
4. Прожимаем ее и у нас появляется mounted disk
5. В папке искать файл прошивки "kbd_firmware/keyboards/crkbd/vial-kb/vial-qmk/.build/crkbd_rev4_1_standard_vial.uf2"
6. !! Сделать бэкап раскладки в vial + залить стандартными средствами на основную половинку. На вспомогательную половинку нужно заливать если прошивка сильно устарела.
